<h2>About 📝</h2>

<p align="center">A project created using JavaScript, HTML and CSS. Basically it's a quiz based in the game "Counter Strike: Global Offensive", the player needs to guess the specific position name of a map in the game! You can select the difficulty (in progress), select the map to train it's positions and view your points when the quiz ends. 👊🏼</p>

<h2>Installation 👨‍💻</h2>
### Before you can start, you have to install this tools in your machine:

- <b>[Git](https://git-scm.com)</b>

### Then:

```
- Clone the repository:
$ git clone https://github.com/ggonpereira/csgo-map-spots-quiz.git

- Then access the directory:
$ cd csgo-map-spots-quiz

- Then, type:
$ index.html
```
